Enter a program that computes a value on the left. The system computes the value on the right. You can then change the value on the right to see the program to change if there is a valid repair.

Supported syntax:
let ID = EXPR in EXPR
 variable evaluation and assignment 
{ label = EXPR, label2 = EXPR }

{ label = EXPR } & { label2 = EXPR }
 records of labelled values (the operator & concatenates records, the comma is syntactic sugar) 
EXPR.label

 extracts the value associated to a label in a record 
"abcd"

EXPR + EXPR
 string values and string concatenation 
EXPR EXPR
 lambda application 
\x. EXPR
λx. EXPR
 lambda abstraction
