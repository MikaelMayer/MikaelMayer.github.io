Embedding Reflex in Blog Posts

Drop a div with a fixed height, add a reflex-formula attribute, and include the loader scripts at the end of the page. Each block becomes a live Reflex canvas.

Host the helper once and reference it via an absolute URL. For GitHub Pages the script will live at https://mikaelmayer.github.io/apps/reflex4you/reflex-embed.js. After tagging 0.1.0 you can also load the immutable jsDelivr copy at https://cdn.jsdelivr.net/gh/MikaelMayer/MikaelMayer.github.io@0.1.0/apps/reflex4you/reflex-embed.js.
